# Scoop Bucket Template

[![Tests](https://github.com/AndreiVernon/scoop-cone/actions/workflows/ci.yml/badge.svg)](https://github.com/AndreiVernon/scoop-cone/actions/workflows/ci.yml) [![Excavator](https://github.com/AndreiVernon/scoop-cone/actions/workflows/excavator.yml/badge.svg)](https://github.com/AndreiVernon/scoop-cone/actions/workflows/excavator.yml)

My bucket for [Scoop](https://scoop.sh), the Windows command-line installer.

## How do I install this bucket?

Run the following:

```pwsh
scoop bucket add scoop-cone https://github.com/AndreiVernon/scoop-cone
scoop install scoop-cone/<manifestname>
```

## How do I contribute new manifests?

To make a new manifest contribution, please read the [Contributing
Guide](https://github.com/ScoopInstaller/.github/blob/main/.github/CONTRIBUTING.md)
and [App Manifests](https://github.com/ScoopInstaller/Scoop/wiki/App-Manifests)
wiki page.
